# Alien Invasion Game

This is a simple 2D game built with python using object-oriented programming(OOP).

## About the Project
In this game, the player controls a spaceship and shoots bullets a spaceship and shoots bullets to destroy incoming
aliens. The goal is to survive as long as possible and score points by hitting aliens.

## Concepts Used
- Object-Oriented Programming
- Game loop
- Collision with detection
- Working with pygame

## Features
- Move the ship left and right
- Shoot bullets
- Alien fleet movement
- Collision detection

  ## How to Run
  1. Install Python
  2. Install Pygame
  3. Run the game
